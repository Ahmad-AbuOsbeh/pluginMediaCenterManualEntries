# pluginMediaCenterManualEntries
BuildFire Plugin: media center with manual entries 

##How to run and test
###Prerequisite are node.js,bower,npm,karma
```bash
$ npm install -g karma-cli
$ npm install -g bower
```
###Install bower and node dependencies
```bash
$ bower install
$ npm install
```
###Run the test cases
```bash
$ npm start
```
